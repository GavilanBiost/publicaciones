# Código y material de apoyo para publicación

[![](https://img.shields.io/badge/DOI-10.1007%2Fs00394--021--02751--5-blue)](https://doi.org/10.1007/s00394-021-02751-5)

## Contenidos

Este directorio contiene:

- **Script de R** para pre-procesar datos y realizar análisis estadísticos
- **Figuras** generadas en el artículo (Figure 1-3.png)
- **PDF** de la publicación (Open Access)

## Requisitos

- **R ≥ 3.6**
- Paquetes utilizados:
  - `haven` - Leer y escribir datos de SPSS, Stata y SAS
  - `readxl` - Leer hojas de cálculo de Excel
  - `Hmisc` - Herramientas estadísticas y gráficas
  - `agricolae` - Análisis de diseño experimental
  - `ggplot2` - Visualización de datos
  - `forestplot` - Gráficos de bosque para meta-análisis
  - `rio` - Importar/exportar datos en múltiples formatos
  - `gvlma` - Validación de supuestos de modelos lineales

## Datos

Por cuestiones de tratamiento y protección de datos, las bases de datos de dichas publicaciones solo son accesibles bajo petición estricta al Steering Committee (SC) de cada estudio respectivo. La forma de contacto con el SC del estudio está disponible dentro de la publicación a través de los autores de correspondencia.

### English

# Code and supporting material for publication

[![](https://img.shields.io/badge/DOI-10.1007%2Fs00394--021--02751--5-blue)](https://doi.org/10.1007/s00394-021-02751-5)

## Contents

This directory contains:

- **R script** for data preprocessing and statistical analysis
- **Figures** generated in the article (Figure 1-3.png)
- **PDF** of the publication (Open Access)

## Requirements

- **R ≥ 3.6**
- Required packages:
  - `haven` - Read and write SPSS, Stata, and SAS data
  - `readxl` - Read Excel spreadsheets
  - `Hmisc` - Statistical and graphics utilities
  - `agricolae` - Experimental design analysis
  - `ggplot2` - Data visualization
  - `forestplot` - Forest plots for meta-analysis
  - `rio` - Import/export data in multiple formats
  - `gvlma` - Validation of linear model assumptions

## Data

For data processing and protection reasons, the databases of these publications are accessible only upon strict request to the Steering Committee (SC) of each respective study. The contact form for the study's SC can be found within the publication through the corresponding authors.
